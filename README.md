Vim plugin for accelerated smooth scroll (mapping to `<C-D>/<C-U>`, `<C-F>/<C-B>`)
--------------------------------------------------------------------------------------

このプラグインは、`<C-D>/<C-U>`, `<C-F>/<C-B>`キーを置き換え、
スムーズなスクロールとスクロール速度の加速を実現するためのプラグインです。


## Installation

```vim
NeoBundle 'yonchu/accelerated-smooth-scroll'
```
